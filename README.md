# 🧮 Calculadora Inteligente em Python

Projeto desenvolvido para praticar estruturas de repetição (`while`) e condicionais (`if/elif`) em Python.

## ✨ Funcionalidades
- Operações básicas: Soma, Subtração, Multiplicação e Divisão.
- Menu interativo.
- Opção de continuar ou sair após o cálculo.

## 🛠️ Tecnologias
- **Python 3.x**

## 🚀 Como rodar o projeto
1. Clone o repositório ou baixe o arquivo `calculadora.py`.
2. Abra o terminal na pasta do arquivo.
3. Digite: `python calculadora.py`
